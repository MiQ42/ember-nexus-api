 Error 403 - Forbidden

 This error is thrown when the requested endpoint, element or action is forbidden.

 There are a few things you can check:

 - Is the endpoint available via the GET method?
 - Do you have access to the endpoint?
 - Is the endpoint disabled in the configuration?

 Link to the documentation: https://ember-nexus.github.io/api

 Link to http.dev's explanation of the 403 error code: https://http.dev/403
