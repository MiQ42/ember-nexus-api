 Error 400 - Incomplete Mutual Dependency

 This error is thrown when there are properties which depend on ech other, i.e. must be all defined, but at least one
 is missing from the request body. Try to add the missing properties.

 Link to the documentation: https://ember-nexus.github.io/api

 Link to http.dev's explanation of the 400 error code: https://http.dev/400
